# Hedelmapeli

Hedelmapeli is an investigation into the elements that make slot machines as playable and addictive as they are.

## How to run locally

```
# Install nvm, then
nvm install 17
nvm use 17

# Install dependencies
npm install
npm start

# and app opens to
http://localhost:3000/

# Make changes and then Ctrl+Shift+R
```
